# Temperature Converter (C++)

A simple command-line program written in C++ that converts temperatures between Fahrenheit, Celsius, and Kelvin.

## What it does

This program lets the user enter a temperature and choose its unit (F, C, or K). It then converts that value into the other two units and displays the results.

## How it works

- The user selects a unit:
  - F for Fahrenheit  
  - C for Celsius  
  - K for Kelvin  
- The user enters a temperature value  
- The program performs the appropriate conversion using standard formulas  
- The results are printed on the screen

## Features

- Converts between all three temperature units
- Simple menu-based input
- Basic input validation for Kelvin (cannot be negative)
- Clean function-based structure
